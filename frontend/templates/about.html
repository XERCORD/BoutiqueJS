Qui sommes-nous ?

Yblazing Fail est une boutique en ligne fictive dédiée aux cartes à collectionner Pokémon et Yu-Gi-Oh!, réalisée dans le cadre du module Challenge JavaScript — Bachelor 1 Informatique.

Nous avons conçu une expérience complète : catalogue, panier, favoris, recherche et API Express + MySQL, avec une interface pensée pour les collectionneurs.
